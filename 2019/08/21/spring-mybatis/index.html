概述

  1. MyBatis是一款优秀的持久层框架
  2. MyBatis支持定制化SQL、存储过程和高级映射
  3. JPA Or MyBatis
    • JPA：数据操作都比较简单
    • MyBatis：DBA需要对SQL进行审核，复杂SQL
  4. Spring + MyBatis
    • MyBatis Spring Adapter
    • MyBatis Spring-Boot-Starter
